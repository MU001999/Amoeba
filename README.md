## Just an exercise for *Compilers*.
> actually just an interpreter, and supports REPL.

The interpreter for the Ice programming language with many bugs.

### Quick start
```shell
git clone https://github.com/MU001999/ice.git
cd ice/ice
make
cd bin
./ice
```
