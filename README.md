# Ice
> this is a implementation that almost by myself in this branch.

A funny language named Ice (an exercise for Compilers).

### Progress

###### Comments
```ice
# this is a comment
```

###### Variable Declaration
```ice
@a: 1
@b: a
```

###### Binary Operator
```ice
# support + - * / % now
@a: 1 + 1
@b: 4 - 1
@c: (a + b) - a * b / a % b # support ()
```
