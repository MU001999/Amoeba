# Ice
A funny language named Ice.

### Progress

###### Comments
```ice
# this is a comment
```

###### Variable Declaration 
```ice
int a
a : 1
int b : a
```

###### Arithmetic Operator
```ice
# support + - * / %
int a : 1 + 1
int b : 2 - 1
(a + b) - a * b / a # also support ()
```

###### print
```ice
int a : 1
int b : a + 3 * 4 - 2 / 2 + 2 % 4
print b # output: 14
```
